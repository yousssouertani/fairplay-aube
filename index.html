Configuration du stockage

L'application utilise JSONBin.io pour stocker les données en ligne, gratuitement. Créez un compte sur jsonbin.io, récupérez votre Master Key, et collez-la ici. Toutes les données seront synchronisées entre tous les appareils.

Sur jsonbin.io : cliquez votre avatar → API Keys → copiez la Master Key.

Comment créer votre clé API gratuite :

1. Allez sur jsonbin.io et cliquez "Sign Up"
2. Créez un compte avec votre email (gratuit, pas de CB)
3. Une fois connecté, cliquez votre avatar en haut à droite
4. Cliquez "API Keys" puis copiez la "Master Key"
5. Collez-la dans le champ ci-dessus et cliquez Sauvegarder
